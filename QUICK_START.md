# Quick Start - Deploy in 5 Minutes! 🚀

## Super Easy Method (No Technical Knowledge Required)

### Step 1: Create GitHub Account (2 minutes)
1. Go to https://github.com
2. Click "Sign up"
3. Enter your email, create a password
4. Verify your email
5. Done! ✅

### Step 2: Create Repository (1 minute)
1. Click the "+" icon in top right
2. Click "New repository"
3. Name it: `world-cup-sweepstakes`
4. Select "Public"
5. **IMPORTANT:** Do NOT check "Add a README file"
6. Click "Create repository"
7. Done! ✅

### Step 3: Upload Files (1 minute)
1. On the repository page, click "uploading an existing file"
2. Drag and drop these files from your computer:
   - `index.html`
   - `script.js`
   - `styles.css`
   - `README.md`
3. Click "Commit changes"
4. Done! ✅

### Step 4: Enable GitHub Pages (1 minute)
1. Click "Settings" tab (top of page)
2. Click "Pages" in left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Done! ✅

### Step 5: Get Your URL
Wait 1-2 minutes, then your site will be live at:
```
https://YOUR-USERNAME.github.io/world-cup-sweepstakes/
```

Replace `YOUR-USERNAME` with your actual GitHub username.
